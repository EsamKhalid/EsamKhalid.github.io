Project overview

Portfolio 1 is the first ever portfolio project that was assigned and submitted, it was a great introduction into what was expected of us as students and how the assignments worked within the Module.

In this portfolio project, we were asked to create a sketch of a vehicle using certain commands in the processing IDE, we were also asked to add an element of randomness to the vehicle, I implemented random color, wheel size and body size.
